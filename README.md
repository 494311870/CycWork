# 作业3：指数纳入效应套利策略的实现

本项目实现了基于中国股票市场数据的指数纳入效应套利策略分析。

## 项目描述

当股票纳入股票指数时，会有正的超额收益。本项目利用沪深300指数2011年1月至2024年8月所有纳入和剔除的股票收益数据，完成以下分析：

1. **问题一**：计算纳入和剔除股票的超额收益率，并构建套利策略
2. **问题二**：寻找与纳入指数股票相匹配的股票（同行业、同市场、Beta相近）
3. **问题三**：利用匹配样本构建配对交易套利策略
4. **问题四**：在不同市场环境（牛市、熊市、非牛熊市）下检验策略稳健性

## 技术细节

### 超额收益率计算

**公式（基于CAPM模型）：**
```
Abret = Dretwd - Beta × Dretwdos
```

**说明：**
- `Abret`: 超额收益率（Abnormal Return）
- `Dretwd`: 个股实际收益率
- `Beta`: 股票的系统性风险系数
- `Dretwdos`: 市场收益率
- `Beta × Dretwdos`: 由市场因素解释的预期收益
- `Abret`: 超出预期的部分，反映指数纳入效应

### Beta匹配算法

**匹配条件（同时满足）：**
1. 同行业：Nnindcd相同
2. 同市场：Markettype相同
3. Beta相近：|Beta - Beta1| < 0.01

**目标函数：**
```
j* = argmin |β(i) - β(j)|
subject to: 同行业、同市场、|Δβ| < 0.01
```

### 套利策略收益

**Mac策略（问题一）：**
```
套利收益 = 纳入股票平均收益率 - 剔除股票平均收益率
```

### 配对交易收益

**Marina策略（问题三）：**
```
Pair_Return(i,j,t) = Dretwd(i,t) - Dretwd(j,t)
Portfolio_Return(t) = (1/N) × Σ Pair_Return(i,j,t)
```

其中N为有效配对数量，采用等权重方法
