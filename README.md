# 作业3：指数纳入效应套利策略的实现

本项目实现了基于中国股票市场数据的指数纳入效应套利策略分析。

## 项目描述

当股票纳入股票指数时，会有正的超额收益。本项目利用沪深300指数2011年1月至2024年8月所有纳入和剔除的股票收益数据，完成以下分析：

1. **问题一**：计算纳入和剔除股票的超额收益率，并构建套利策略
2. **问题二**：寻找与纳入指数股票相匹配的股票（同行业、同市场、Beta相近）
3. **问题三**：利用匹配样本构建配对交易套利策略
4. **问题四**：在不同市场环境（牛市、熊市、非牛熊市）下检验策略稳健性

### ArbitrageStrategyAnalyzer 类

主要方法：

- `solve_question1()`: 计算纳入和剔除股票的超额收益率，构建Mac套利策略

## 分析结果

### 问题一：指数纳入效应

- 计算了纳入和剔除股票在公告日及其后5天的超额收益率
- 统计分析显示存在显著的指数纳入效应和剔除效应
- Mac的等权重策略在某些时间窗口能获得显著收益

### 问题二：匹配样本

- 根据Beta值匹配精度<1%的要求，为每个纳入指数的股票找到匹配股票
- 匹配考虑了同行业、同市场等因素

### 问题三：配对交易策略

- 实现了基于5对匹配样本的配对交易策略
- 计算了10个交易日的Buy & Hold收益率
- 生成了收益率曲线图以直观展示策略表现

## 技术细节

### 超额收益率计算

```
超额收益率 = Dretwd - Beta × Dretwdos
```

其中：
- `Dretwd`: 个股收益率
- `Beta`: 股票的系统性风险
- `Dretwdos`: 市场收益率

### 套利策略收益

```
套利收益 = 纳入股票收益率 - 剔除股票收益率
```

### 配对交易收益

```
配对收益 = 纳入股票收益率 - 匹配股票收益率
```

## 结论

1. 中国股票市场存在显著的指数纳入效应
2. 配对交易策略通过对冲市场风险，能在不同市场环境下保持相对稳定的表现
3. 策略在不同市场环境下的表现存在差异，需要根据市场环境调整策略参数
